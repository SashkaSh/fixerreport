# FixerReport app 

The Fixer Report app generates a report on currency development.

## Development Requirements

  * Ruby 2.7.0+
  * Bundler

## Installation

```
rvm install 2.7.0
rvm use 2.7.0
bundle install
```

Or you can use Docker.

## Genarate report:

`$ rake generate_report`
